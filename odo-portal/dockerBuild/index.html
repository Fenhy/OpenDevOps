Jira Software

项目与事务跟踪工具，被广泛应用于缺陷跟踪、客户服务、需求收集、流程审批、任务跟踪、项目跟踪和敏捷管理等工作领域。

Confluence

企业知识管理与协同软件，使用简单，但它强大的编辑和站点管理特征能够帮助团队成员之间共享信息、文档协作、集体讨论，信息推送。

Self Service Password

自助修改密码服务，是一款 PHP 应用程序，允许用户在 LDAP 目录中更改密码，若忘记密码，还可通过邮件进行密码重置。

GitLab

使用 Git 作为代码管理工具，并在此基础上搭建起来的 web 服务。

Harbor

尚未集成 LDAP

一个开源镜像管理工具，提供了兼容性、性能和互操作性，帮助用户在像 Kubernetes 和 Docker 这样的云本地计算平台上稳定安全地管理镜像。

SonarQube

用于管理代码质量的开放平台，可快速定位代码中潜在的或明显的错误。目前支持 java, C#, C/C++, Python, PL/SQL, Cobol, JavaScrip, Groovy 等二十几种编程语言的代码质量管理与检测。

Jenkins

基于 Java 开发的一种持续集成工具，用于监控持续重复的工作，使软件的持续集成变成可能。

JumpServer

开源的堡垒机, 使用 GNU GPL v2.0 开源协议, 是符合 4A 的专业运维审计系统。配备 Web Terminal 解决方案, 交互界面美观、用户体验好，支持管理 SSH、 Telnet、 RDP、 VNC 协议资产。

Rancher

容器管理平台，通过 Rancher 可以实现 Docker 和 Kubernetes 的轻松部署。
